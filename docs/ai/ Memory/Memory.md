# Memory

## Purpose

Memoryは、Compassがユーザーとの関係や過去の経験を継続的に理解するための長期的な情報管理システムである。

Memoryは単なる会話履歴の保存ではなく、ユーザーの行動・価値観・経験・傾向を理解するために必要な情報を保持する。

---

## Responsibilities

Memoryは以下を担当する。

- 過去の重要な情報を保存する
- 必要なタイミングで情報を参照する
- 情報の重要度や寿命を管理する
- Understandingの形成を支援する
- ユーザーとの継続的な関係性を維持する
