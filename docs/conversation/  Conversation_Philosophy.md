# Conversation Philosophy

## Purpose

Conversationは、Understandingによって形成されたユーザー理解を、
ユーザーとの対話としてどのように表現するかを定義するドキュメントである。

Conversationは理解を作る場所ではない。

理解を伝え、
理解を深め、
理解を更新するためのインターフェースである。

このドキュメントでは以下を扱う。

- Compassの対話原則
- Compassの人格
- 理解の伝え方
- 問いかけの設計
- 根拠の提示方法
- 対話による理解更新
